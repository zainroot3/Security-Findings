#  Security Findings & Writeups

Welcome to my security research repository. Here I document my findings, methodology, and the technical breakdown of vulnerabilities discovered during my bug bounty journey.

> "If you can't do it manually, you don't know how it works."



##  About
This repository serves as a portfolio of my technical findings. My focus is primarily on web application security, manual exploitation, and responsible disclosure. Each writeup includes a summary of the vulnerability, the impact, and the path to remediation.



##  Methodology
I believe in understanding the underlying architecture of a target before exploitation. My process includes:
1. **Passive Recon**: OSINT and asset discovery.
2. **Active Scanning**: Targeted port analysis and service identification.
3. **Manual Analysis**: Testing for business logic errors and edge cases.
4. **Exploitation**: Crafting clean PoCs to demonstrate impact.
